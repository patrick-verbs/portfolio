"Cupcake Shop"

Live webpage | Repository
A pair-programming exercise done in Week 1 of the Epicodus full-time, full-stack developer bootcamp. The assignment was to create a "cupcake shop" web page using three image elements. I created the file structure, sourced the images, and wrote most of the HTML and CSS during my 15 minutes of driving (having my turn at the keyboard and mouse).


"Animal Shelter"

Live webpage | Repository
Another pair-programming exercise made during Week 1 of the Epicodus developer program. This was to be a simple mockup of an "animal shelter" website using basic HTML and CSS, including a sidebar navigation element.


"Wikipedia Rebuild"

Live webpage | Repository
This was a Bootstrap exercise, also done as an optional assignment during the first week of the Epicodus bootcamp. The goal was to recreate Wikipedia's Main Page using Bootstrap; as I wasn't familiar with Bootstrap, I limited my focus to recreating the Main Page's grid layout using the CSS framework's "container," "row," and "col" elements.
